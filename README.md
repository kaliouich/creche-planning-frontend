# Crèche Planning - Frontend

Ceci est l'interface utilisateur de l'application de gestion des plannings de permanence pour une crèche associative.

## Fonctionnalités
- **Interface Administrateur** : Création des semaines, gestion des créneaux, suivi des saisies des parents (disponibilités, enfants présents/absents), génération automatique du planning, et statistiques.
- **Interface Parent** : Saisie facile et rapide des disponibilités pour les permanences, et consultation du planning publié.
- **Design dynamique** : Interface moderne, fluide, et responsive.

## Technologies
- React
- Vite
- TypeScript
- Tailwind CSS (via index.css personnalisé)
- Lucide React (Icônes)

## Installation & Lancement

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```

L'application sera accessible (par défaut) sur `http://localhost:5173`.
